Desafio das Cores

O Desafio das Cores é muito simples, você deve apenas clicar no botão com a cor correspondente ao alvo. O alvo sempre será a cor da palavra escrita na tela.

Clique na cor correta antes que o tempo acabe para receber seu próximo alvo. Seus pontos são computados a cada acerto e termina se você demorar muito ou escolher a cor errada.
